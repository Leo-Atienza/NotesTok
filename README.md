# NotesTok

Your Notes Become Your Study Companion

AI-powered study companion that transforms any study material into interactive micro-lessons with forced active recall quizzes. Built for Build With AI 2026 (GDG UTSC).

## Setup

```bash
npm install
cp .env.example .env.local
# Add your Gemini API key to .env.local
npm run dev
```

## Tech Stack

- Next.js 15 + TypeScript + Tailwind CSS + shadcn/ui
- Google Gemini 2.5 Flash (content analysis, lesson generation, adaptive tutoring)
- Browser Web Speech API (TTS narration)

Powered by Google Gemini.
